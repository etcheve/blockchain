# mini-blockchain web

A visual blockchain explorer built with React + Vite.

## Prerequisites

- Node.js (v18+ recommended)

## Install dependencies

```bash
cd mini-blockchain/web
npm install
```

> Skip this step if `node_modules/` already exists.

## Run in development

```bash
npm run dev
```

Opens at `http://localhost:5173` (Vite default).

## Build for production

```bash
npm run build
```

Output goes to `dist/`.

## Preview production build

```bash
npm run preview
```
