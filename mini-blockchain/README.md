# mini-blockchain

A minimal educational blockchain implemented in two languages with identical logic.

| | C++ | Rust |
|---|---|---|
| SHA256 | OpenSSL | sha2 crate |
| Storage | `std::vector<Block>` | `Vec<Block>` |

## Structure

```
toy-blockchain/
├── cpp/
└── rust/
```

See each subfolder's README for build instructions.
